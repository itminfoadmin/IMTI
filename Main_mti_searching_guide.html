단계별 상세 설명
🔍
Step 01 · 필터 조건 선택
SEARCH FILTER

상단 검색바에서 영역(D)·대상(P)·지향(O)·역량(C) 중 원하는 조건을 클릭해 드롭다운에서 선택하세요. 하나 이상 선택하면 검색 결과 드롭다운에 검색 결과가 표시됩니다.

1개 이상 선택 필수
📋
Step 02 · 수업 모형(MTI) 선택
SELECT MODEL

필터 조건을 선택하면 오른쪽 결과 칸이 분홍색으로 바뀌며 검색된 수업 모형(MTI)의 건수가 표시됩니다. 결과 칸을 클릭하면 해당 수업 모형 목록이 드롭다운으로 펼쳐집니다.

분홍 배경 = 결과 있음
📄
Step 03 · 수업 모형(MTI) 확인
VIEW DOCUMENT

드롭다운에서 수업 모형(MIT)을 선택하면 아래 뷰어에 해당 수업 모형(MTI)의 상세 내용이 바로 표시되며, 오른쪽의 사이드 버튼을 이용하여 수업 모형(MTI)에 대한 부가기능을 이용하실 수 있습니다.

선택 즉시 문서 전환
🔄
필터 초기화 (Reset)
RESET ALL

검색바의 오른쪽 끝 Reset ↺ 버튼을 클릭하면 모든 필터와 선택된 수업 모형(MTI)이 초기화되어 이 페이지로 돌아옵니다. 다른 조건으로 다시 검색할 때 사용하세요.

조건 변경 시 활용


필터키 상세 설명
필터 키 차원(Dimention) 선택 가능 항목
영역(D) 선택 ▾ 정보 교과 영역
(Doamins)		 컴퓨터 시스템/ 데이터/ 알고리즘과 프로그래밍/ 인공지능(AI)/ 디지털 문화/ 기타
대상(P) 선택 ▾ 대상 학습자 유형
(Participants)		 영유아/ 초등학생/ 중학생/ 고등학생/ 특수학습자/ 영재학습자/ 성인학습자
지향(O) 선택 ▾ 지향된 학습 형태
(Orientations)		 문제 해결/ 과제 수행/ 공동 학습/ 심화 학습/개념 학습/ 안내된 실습/
역량(C) 선택 ▾ 기대되는 역량
(Competencies)		 컴퓨팅 사고력/ 디지털 문화 소양/ 인공지능 소양/기타
💡

필터 키 활용 팁. 필터 조건을 여러 개 조합할수록 결과가 좁혀집니다. 원하는 수업 모형(MTI)이 보이지 않을 때는 조건을 하나씩 줄여보세요.
결과 드롭다운에서 항목에 마우스를 올려두면 해당 문서가 미리 로딩되어, 클릭 시 더 빠르게 표시됩니다.
