단계별 상세 설명
🔍
Step 01 · 필터 조건 선택
SEARCH FILTER

상단 검색바에서 영역(D)·대상(P)·지향(O)·역량(C) 중 원하는 조건을 클릭해 드롭다운에서 선택하세요. 하나 이상 선택하면 검색 결과 드롭다운에 검색 결과가 표시됩니다.

📋
Step 02 · 수업 모형(MTI) 선택
SELECT MODEL

필터 조건을 선택하면 오른쪽 결과 칸이 하늘색으로 바뀌며 검색된 수업 모형(MTI)의 건수가 표시됩니다. 결과 칸을 클릭하면 해당 수업 모형 목록이 드롭다운으로 펼쳐집니다.

📄
Step 03 · 수업 모형(MTI) 확인
VIEW DOCUMENT

드롭다운에서 수업 모형(MTI)을 선택하면 아래 뷰어에 해당 수업 모형(MTI)의 상세 내용이 바로 표시되며, 오른쪽의 사이드 버튼을 이용하여 수업 모형(MTI)에 대한 부가기능을 이용하실 수 있습니다.

🔄
필터 항목 초기화
CLEAR FILTER

각 필터(영역·대상·지향·역량) 라벨 옆의 ✕ 버튼을 클릭하면 해당 조건만 선택 해제됩니다. 다른 조건은 그대로 둔 채 특정 조건만 다시 선택하고 싶을 때 사용하세요.


부가 기능
✏️
나의 경험 공유하기
선택한 수업 모형을 실제 수업에 적용한 경험을 공유합니다. 수업 대상, 내용, 좋았던 점, 문제점, 조언 등을 입력한 뒤 등록하세요.
📋
수업 모형 적용 사례
해당 모형을 실제 수업에 적용한 사례와 후기를 확인할 수 있습니다. 아직 사례가 없으면 첫 경험 공유 팝업으로 연결됩니다.
🔗
RISS 바로 가기
학술 논문·연구 자료 검색 사이트인 RISS를 새 탭으로 바로 열어줍니다. 관련 연구물을 직접 조회할 때 활용하세요.
⬇️
PDF 다운로드
현재 보고 있는 수업 모형(MTI) 문서를 PDF 파일로 다운로드합니다. 버튼을 누르면 다운로드 여부를 묻는 확인창이 표시되며, "예"를 선택하면 다운로드가 진행됩니다.
