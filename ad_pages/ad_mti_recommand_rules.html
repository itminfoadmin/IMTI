Priority Weights RULE 01
각 조건의 우선순위와 일치도(%) 계산에서 차지하는 최대 배점(가중치)을 설정합니다. 우선순위는 추천 결과를 정렬할 때 어떤 조건의 일치를 더 우선해서 볼지를 결정하며(1순위가 가장 높음), 가중치는 네 값의 합으로 정규화하여 0~100% 점수를 산출합니다.
가중치 합계 0 —
Defaults & Thresholds RULE 02
사용자가 조건을 선택하지 않았을 때 부여하는 기본 친화도(0~1)와, 결과 노출을 위한 최소 일치도/최대 표시 개수를 설정합니다. 기본 친화도는 해당 조건 가중치에 곱해져 점수로 환산됩니다.
Affinity Chains RULE 03
완전히 일치하지 않더라도, 같은 체인 안에서 가까이 위치한 항목끼리는 부분 점수를 부여합니다. 체인 내 거리가 가까울수록(서로 인접할수록) 친화도가 1에 가까워집니다. 순서가 곧 유사도이므로 항목 배치 순서가 중요합니다.
Option Layout & Order RULE 04
추천 화면에서 각 조건의 선택지가 노출되는 1열/2열 배치와 순서입니다. 여기에 정의되지 않은 새 항목은 데이터에서 발견되는 대로 2열 끝에 가나다순으로 자동 추가됩니다.
